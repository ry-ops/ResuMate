## Common Questions

**Q: How long does parsing take?**
A: Usually 5-10 seconds depending on PDF size.

**Q: How many headlines are generated?**
A: 5-7 options with different styles and focuses.

**Q: What's the character limit for summaries?**
A: 2,000 characters (LinkedIn's limit).

**Q: Do I need an API key?**
A: Yes, you need a Claude API key set in ATSFlow.

**Q: Can I edit the generated content?**
A: Yes! All generated content can be copied and edited manually.

**Q: Is my data stored anywhere?**
A: No, all processing is client-side or via secure API. Nothing is permanently stored.

## Troubleshooting

**Problem: PDF won't upload**
- Solution: Ensure it's a genuine LinkedIn PDF export
- Check file isn't corrupted
- Try re-exporting from LinkedIn

**Problem: Headlines are too long**
- Solution: They auto-truncate to 120 chars
- Try shorter role names or fewer skills

**Problem: API error**
- Solution: Check your API key is set
- Verify internet connection
- Review browser console for details

**Problem: Clipboard copy doesn't work**
- Solution: A fallback method should activate automatically
- Check browser permissions
- Use HTTPS if possible
